## Quik Start

1. Adapt the environment to Development Environment sectoin.
2. Run `npm i` command
3. Run `npm run start` command
4. Connect Nintendo Switch Pro Controller to PC by Bluetooth or USB cable.
5. Play electron-phaser-desktop-game.
